MyArrow-Server:


Der MyArrow-Server ist die Serverkomponente zum entsprechenden Client auf dem Handy. Die Verbindung funktioniert nur bei bekannten IP Adressen und im gleichen Netz.

Die Aufgabe ist es, die Daten zwischen Handy und Server zu synchronisieren und auf alle Clients zu verteilen.

I'm a Simple Form

Enter some text and click the Submit button.
Clicking Submit invokes MyArrowServlet.java,
which returns an HTML page to the browser.
